Projetos em Análise de Dados

Análise de desempenho de hipermercado.

A empresa varejista busca superar desafios relacionados à maximização de desempenho e otimização de recursos em um mercado competitivo. A ausência de uma análise detalhada sobre o comportamento de compra dos clientes, sazonalidade das vendas e rentabilidade por categoria e região tem limitado a eficácia de estratégias de marketing, gestão de estoque e logística. Este projeto visa preencher essas lacunas, fornecendo insights estratégicos para decisões baseadas em dados.

Ferramentas utilizadas

  • Power BI: Para ETL e construção de dashboards.
  • Git e GitHub: Para controle do código
  • Dashboard
  • GitHub

Análise de vício em redes sociais.

"Este projeto de análise de dados aprofundou-se no complexo relacionamento entre o uso de mídias sociais e o bem-estar geral de estudantes, com foco na saúde mental, dependência e qualidade do sono, além de sua correlação com o desempenho acadêmico. Utilizando dados abrangentes com 705 estudantes de 110 nacionalidades distintas, a análise buscou ir além do mapeamento de padrões de uso, visando uma compreensão multidimensional do impacto das interações digitais.
